# Project Rules for AI Agents

- **Always Deploy Live**: After any significant code changes or fixing a bug, always commit and push the changes to GitHub to trigger a live deployment.
- **Never Use the Browser**: Do not use the `browser_subagent` or any browser-related tools to verify changes or search for info. Use code analysis, terminal tools, and local verification methods instead.
- **Automatic Execution**: If you are fixing something or completing a task, proceed with the deployment step without asking for explicit permission.
